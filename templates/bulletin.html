Project Description

Bulletin is a digital pin board built using the Django python framework. You have the ability to sign up, make posts, edit posts, delete posts, and view posts made by others. You can also set a profile picture and a biography. In this way it is a rudimentary social media platform. It is themed to be a bulletin board where all the posts look like sticky notes.


Bulletin was the first of 2 projects I built during my senior year of highschool. The Programming 12 class offered at my highschool had the exact same lesson plan and assignements as the Programming 11 class I had taken the year prior. Instead of just resubmitting all my previous work for an easy grade, I wanted to challenge myself and learn. After clearing it with my teacher, I got to work building Bulletin, and later Harmony.
