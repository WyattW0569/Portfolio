Project Description

SET (short for Smart Education Technologies) was my group's term project for the Engr120 design portion during the spring 2024 semester. The goal of the project was to design a smart-system using a Raspberry Pi Pico-W and the Micro Python programming language to improve some aspect of society. This project had to have both a physical element and webapp compainion. My group received a mark of 100%.


My group's project was essentially a smart HVAC system for schools. Whilst we all contirbuted to every aspect of the project, my focus was mainly on the webapp frontend and backend, and smoothly integrating the sensor code into the webapp. For a more detailed look at this project, plese watch the video below
